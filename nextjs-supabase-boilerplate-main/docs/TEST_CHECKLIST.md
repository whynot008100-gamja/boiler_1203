### 5. 에러 케이스 테스트

- [ ] 로그인하지 않은 상태에서 장바구니 접근 시 로그인 페이지로 리다이렉트
- [ ] 로그인하지 않은 상태에서 주문 접근 시 로그인 페이지로 리다이렉트
- [ ] 존재하지 않는 상품 ID 접근 시 에러 처리
- [ ] 존재하지 않는 주문 ID 접근 시 에러 처리
- [ ] 장바구니가 비어있을 때 주문하기 시 에러 처리
- [ ] 결제 실패 시 에러 메시지 표시
- [ ] 네트워크 오류 시 적절한 에러 메시지 표시
