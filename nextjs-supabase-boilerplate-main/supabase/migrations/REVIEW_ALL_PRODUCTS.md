1. 데이터베이스에 실제로 "클린코트"라는 의류 상품이 도서 카테고리에 잘못 저장되어 있을 수 있습니다.
2. 또는 "클린 코드"를 "클린코트"로 잘못 읽으셨을 수 있습니다.

## 🔍 확인 방법

Supabase Dashboard → SQL Editor에서 다음 쿼리 실행:

```sql
-- "클린"이 포함된 모든 상품 확인
SELECT 
  id,
  name,
  category,
  description,
  price
FROM products
WHERE name LIKE '%클린%'
ORDER BY name;
```

## 📝 카테고리별 상품 수 확인

```sql
SELECT 
  category,
  COUNT(*) as product_count
FROM products
WHERE is_active = true
GROUP BY category
ORDER BY category;
```
